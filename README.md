#  Retail Sales Analysis (SQL + Power BI)

## About this project

This is one of my first data analysis projects. The goal was to take a retail dataset, clean it using SQL, and then build a dashboard in Power BI to understand sales performance.

I wanted to practice working with real-looking business data and answering basic business questions.

---

## What I did

* Cleaned the dataset in SQL (renamed columns, fixed date formats, checked for missing values)
* Wrote SQL queries to explore the data
* Created a Power BI dashboard to visualize the results

---

## Tools I used

* SQL (MySQL)
* Power BI

---

## What I analyzed

Some of the questions I tried to answer:

* Which products generate the most revenue?
* How do sales change over time?
* Which categories are most profitable?
* Who are the top customers?

---

## Dashboard

![Dashboard](images/dashboard_screenshot.png)

The dashboard includes:

* Total revenue, profit, and number of orders
* Monthly sales trend
* Top products
* Sales by category
* Customer overview

---

## Key things I learned

* How to clean and prepare data using SQL
* How to use aggregation queries (SUM, GROUP BY, etc.)
* How to turn data into simple visuals in Power BI
* How to think about data from a business perspective
